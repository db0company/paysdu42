Emacs

Débuter avec Emacs

  • Ouvrir emacs : Taper emacs (ou ne si vous êtes à epitech) dans votre terminal.
  • Mettre un header epitech : Ctrl + C + Ctrl + H
  • Fermer emacs : Ctrl + X + Ctrl + C
  • Ouvrir un fichier : emacs "chemin du fichier"
  • Ouvrir un fichier dans emacs déjà ouvert : Ctrl + X + Ctrl + F
  • Changer le mode d'emacs : Alt + X 'mode'-mode (exemple : C-mode, html-mode, ect)
  • Annuler ce que je suis en train de demander : Ctrl + G
  • Annuler (revenir en arrière) : Ctrl + /
  • Ne pas utiliser en mode graphique : emacs -nw

Partager la fenêtre emacs

  • Séparer la fenêtre en 2 morceaux : Ctrl + X + 3
  • Aller à l'autre fenêtre : Ctrl + X + O
  • Supprimer une fenêtre : Ctrl + X + 0

Fonctionnalités sympas

  • Rechercher un mot : Ctrl + S
  • Remplacer un mot par un autre : Alt + X + 'replace-string'
  • GDB dans emacs : Alt + X + 'gdb'. A ouvrir avec le partage de fenêtre pour voir les étapes une par une dans le fichier source.
  • Parcourir un dossier : Ctrl + X + D
  • Indenter à la norme (uniquement à l'école) : Ctrl + C + Ctrl + Q
