Recuperer un pdf sur l'intra

Exemple : les maths. On n'arrive jamais a voir les cours ou les exos.

Les exos : Clique droit sur le lien du pdf et fais 'Enregistrer la cible du lien sous'
ou 'Save Link as...'. Ouvre le fichier avec KPDF ou KGhostview.

Les cours : Ouvrir Opera (clic droit/Net/Browsers/opera). Aller a la page du cours.
Dans la petite barre 'Etape', choisir le cours ou les exos. Il propose
d'enregistrer le pdf. Ouvre le fichier avec KPDF ou KGhostview.
Si vous etes des adeptes de firefox et que vous ne voulez pas changer vos habitudes, allez dans les options, puis application et choisissez "always ask" pour les pdf.
